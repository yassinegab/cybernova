# Cybernova

Cybernova is a 2D adventure game built in C with SDL 1.2. Players explore a neon-lit futuristic city seized by an AI uprising, navigating immersive levels filled with puzzles, hazards, and cooperative challenges.

## 🌟 Highlights

- **Expansive futuristic city** with varied districts, custom puzzles, enemy encounters, and environmental obstacles.
- **Three richly designed levels** that combine platforming, strategy, and exploration.
- **A climactic boss battle** that brings the adventure to a dramatic conclusion.
- **Advanced gameplay systems** such as a resizable mini-map, double jump, shielding mechanics, and integrated mini-games.
- **Cooperative multiplayer support** with keyboard and game controller controls for shared gameplay.

## 🛠️ Built With

- C programming language
- SDL 1.2 library
- Developed on Ubuntu
